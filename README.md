# ☕ Seadanya Kopi

Website landing page UMKM **Seadanya Kopi** dengan desain modern untuk menampilkan menu kopi, lokasi, dan fitur pemesanan sederhana.

## 🚀 Fitur
- Landing page modern
- Tampilan menu kopi
- Informasi lokasi
- Tombol pesan sekarang
- Responsive design
- Navigasi sederhana

## 🛠️ Teknologi Yang Digunakan
- HTML
- CSS
- JavaScript

## ▶️ Cara Menjalankan Project
1. Download / clone repository ini
2. Buka folder project
3. Jalankan file `index.html`
4. Website siap digunakan

## 🎯 Tujuan Project
Project ini dibuat untuk:
- Website UMKM kopi
- Portfolio web developer
- Landing page sederhana
- Template website coffee shop
